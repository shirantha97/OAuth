## Run Locally

- Clone your forked repository
    ```
    git clone https://github.com/USERNAME/OAuth
    cd OAuth
    ```
- Install all the dependencies
    ```
    npm install
    ```
- Start the server with `npm start`

- Visit your app at http://localhost:5000
